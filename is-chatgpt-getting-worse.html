The Verdict: Yes, It's Getting Worse

Thousands of users report the same problems. You're not imagining it. ChatGPT's quality has measurably declined since GPT-4o, and each new version seems to make it worse.

Common Symptoms Users Report

  • Shorter, lazier responses - Where it once gave detailed explanations, now you get brief, surface-level answers
  • Memory problems - Forgets context mid-conversation, even in the same chat session
  • Refuses more requests - More "I can't help with that" responses for harmless queries
  • Generic, corporate tone - Lost its personality and now sounds like a compliance-trained HR bot
  • Math errors - Can't reliably add numbers or do basic calculations
  • Hallucinations - Makes up facts, citations, and quotes with complete confidence
  • Slower performance - Longer wait times, more "network error" messages
  • Lost creative ability - Writing is flat, unimaginative, and repetitive

Why Is This Happening?

Cost Cutting

Running AI models is expensive. OpenAI may be reducing computational resources per query to save money, resulting in shorter, less thorough responses. Users have called it "AI shrinkflation."

Over-Moderation

Each update adds more safety guardrails. While some moderation is reasonable, users report the model now refuses harmless requests and hedges everything with excessive disclaimers.

Training on AI-Generated Content

As the internet fills with AI-generated text, models trained on this data may degrade over time. Researchers call this "model collapse" or "AI brain rot."

Prioritizing Speed Over Quality

Faster responses mean lower costs, but also less "thinking time" for the model. The result: quicker but dumber answers.
